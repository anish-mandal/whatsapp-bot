# Whatsapp Chat Bot

> powered by [Alpaca LoRA](https://github.com/tloen/alpaca-lora)

## Features

- Replay to messages ( with prefix: "!~" ) automatically.

## Run Locally

Clone this repository:

```bash
git clone https://github.com/anish-mandal/whatsapp-bot.git
```

Install packages:

```bash
yarn install
```

Start:

```bash
yarn start
```
